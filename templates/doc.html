Domocontrol documentation

Installation

Domocontrol Version TWO (first version demo is available at http://my-tek.no-ip.org:8080 and on github at address: https://github.com/DomoControl/domocontrol) require:
  • Raspberry PI board
  • One or more board interface with more inputs and outputs to control home devices.
  • python language
  • Flask Framework with plugins
  • Sqlite3 to store informations
  • In second stage, more slave board I/O with Arduino connected with Raspberry by RS485

Main Menu

  • Home: project presentation
  • Status: shows the status of the I/O
  • Stup
  • Login / Logout
  • Log: user access
  • Doc: software installation and user manual
